## 컬러: Option A (Stone + Indigo)

> 상세 CSS 변수, Tailwind config, WCAG 검증은 **03_테마시스템.md** 참조

| 축 | 선택 | 이유 |
|----|------|------|
| Gray | Stone (갈색 틴트) | 따뜻함, AI 느낌과 거리가 멂 |
| Primary | Indigo (보라 기운) | 지적, 차별화 |
| Accent | Emerald (초록) | 따뜻함, Indigo 보색 대비 |
| Dark BG | Stone 900 | 순수 검정 아님, 따뜻한 어둠 |

모든 hex는 Tailwind v3 공식 팔레트. 커스텀 hex 없음.

**사용 규칙:** 컴포넌트 hex 하드코딩 금지. Accent과 Success 혼용 금지. Warning(Amber) 라이트 배경 텍스트 금지. 프로그레스 바 2+1색 제한.

## 상태 패턴

**Skeleton:** Light Stone200→100 / Dark Stone600→700, pulse 1.5s.
**Empty State:** 중앙정렬, Lucide 48px(Light Stone400 / Dark Stone500) + 메시지 + CTA.
**Error:** API실패→Danger카드+재시도 / 네트워크→Warning배너 / 인증만료→Toast.
**Toast:** 우측하단, 3개, 5초, z-60. Success/Info/Warning/Error 좌측보더.

## 접근성

Focus Ring: 2px solid var(--primary), offset 2px. 터치 타겟: 44x44px. 스크린 리더: aria-label, aria-live. WCAG AA: 모든 텍스트/배경 4.5:1+.
