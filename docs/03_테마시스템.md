## 금지 규칙

1. 컴포넌트 hex 하드코딩 금지
2. `bg-stone-100` 대신 `bg-raised` 사용
3. `dark:bg-stone-800` 대신 `bg-surface` (자동 스위칭)
4. Accent과 Success 혼용 금지
5. Warning 라이트 배경 텍스트 금지 (Amber 800 대체)
