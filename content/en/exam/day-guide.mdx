## ProctorFree Environment Setup

CCA-F uses ProctorFree online proctoring. Verify your setup well before exam day.

### Checklist

| Item | Requirement | Check |
|------|------------|-------|
| OS | Windows 10+, macOS 10.15+, ChromeOS | ☐ |
| Browser | Chrome latest version (recommended) | ☐ |
| Webcam | Built-in or external webcam required | ☐ |
| Microphone | Built-in or external mic required | ☐ |
| Internet | Stable wired connection recommended (min 2Mbps) | ☐ |
| Room | Quiet private space, no other people | ☐ |
| Desk | Clean desk, no exam-related materials | ☐ |
| Monitor | Single monitor only (no dual monitors) | ☐ |

### Pre-Test

Run the ProctorFree system check 24 hours before your exam:

1. Click "System Check" on the exam link
2. Approve webcam, microphone, and screen sharing permissions
3. Verify network speed test passes
4. Disable browser extensions (especially VPN, ad blockers)

## ID Requirements

- **Passport** (most recommended — includes English name)
- International driver's license (English)
- Government-issued photo ID in English

## Time Management

120 minutes for 60 questions — average **2 minutes** per question.

### Recommended Pacing

| Segment | Time | Strategy |
|---------|------|----------|
| Q1-20 | 40 min | Quick pass on foundational concepts |
| Q21-40 | 40 min | Careful analysis of scenario questions |
| Q41-60 | 30 min | Remaining questions + challenging items |
| Review | 10 min | Revisit marked questions |

### Key Strategies

- **Mark for Review**: Skip uncertain questions and return later
- **Elimination**: Remove clearly wrong answers first
- **Read the question first**: For long scenarios, read the question before the passage
- **Pace check**: 30 questions done within 60 minutes is on track

## Technical Issues During the Exam

### Internet Disconnection

1. Stay calm — most cases auto-reconnect
2. Do not close the browser
3. Exam resumes after network recovery
4. Timer continues, so quick recovery is important

### Browser Crash

1. Reopen the exam link — progress is restored
2. Answers are auto-saved on the server
3. Contact ProctorFree support via chat immediately

### Proctor Connection Issues

1. If wait exceeds 15 minutes, request chat support
2. If proctor connection fails before exam start, rescheduling is possible
