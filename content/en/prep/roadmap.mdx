## 12-Week Standard Roadmap

If you have time, the 12-week roadmap lets you cover each domain in depth with ample practice.

| Week | Topic | Key Activities |
|------|-------|---------------|
| 1-2 | Foundations | Understand Claude ecosystem, read Exam Guide PDF, map domain structure |
| 3-4 | Agentic Architecture (27%) | Agent loops, orchestration, multi-agent patterns |
| 5-6 | Claude Code & Prompt Engineering (40%) | CLAUDE.md, slash commands, JSON Schema, few-shot |
| 7-8 | Tool Design & MCP + Context (33%) | MCP servers, tool design, context management |
| 9-10 | Scenario Analysis | Walk through all 6 scenarios, practice domain mapping |
| 11 | Mock Exam & Review | Full mock exam, wrong answer analysis, weakness remediation |
| 12 | Final Check | Cheatsheet review, system test, exam day |

## 2-4 Week Fast Track

For experienced developers already familiar with AI tools and Claude.

### 2-Week Route (2-3 hours per day)

| Day | Activity |
|-----|----------|
| Day 1-2 | Read Exam Guide PDF + all domain overviews |
| Day 3-4 | Deep dive on Agentic Architecture (27% — highest weight) |
| Day 5-6 | Claude Code + Prompt Engineering |
| Day 7-8 | MCP + Context Management |
| Day 9-10 | All 6 scenario walkthroughs |
| Day 11-12 | Cheatsheet review + mock exam |
| Day 13-14 | Wrong answer analysis + take the exam |

### 4-Week Route

Based on the 2-week route with an extra day per domain for deeper study.

## Study Strategy Tips

1. **Allocate time by weight**: Spend the most time on Agentic Architecture (27%)
2. **Practice scenario-domain mapping**: Know which domains each scenario covers
3. **Learn anti-patterns**: "What NOT to do" questions appear frequently
4. **Get hands-on experience**: Theory alone isn't enough — use Claude Code directly
5. **Read questions first**: For scenario-based questions, read the question before the long scenario text
