## ProctorFree 환경 세팅

CCA-F 시험은 ProctorFree 온라인 감독 시스템을 사용합니다. 시험 전에 반드시 환경을 확인하세요.

### 체크리스트

| 항목 | 요구 사항 | 확인 |
|------|----------|------|
| OS | Windows 10+, macOS 10.15+, ChromeOS | ☐ |
| 브라우저 | Chrome 최신 버전 (권장) | ☐ |
| 웹캠 | 내장 또는 외장 웹캠 필수 | ☐ |
| 마이크 | 내장 또는 외장 마이크 필수 | ☐ |
| 인터넷 | 안정적인 유선 연결 권장 (최소 2Mbps) | ☐ |
| 방 환경 | 조용한 개인 공간, 타인 출입 불가 | ☐ |
| 책상 | 깨끗한 책상, 시험 관련 자료 없음 | ☐ |
| 모니터 | 듀얼 모니터 사용 불가 (1대만) | ☐ |

### 사전 테스트

시험 24시간 전에 ProctorFree 시스템 테스트를 실행하세요.

1. 시험 링크에서 "System Check" 클릭
2. 웹캠, 마이크, 화면 공유 권한 승인
3. 네트워크 속도 테스트 통과 확인
4. 브라우저 확장 프로그램 비활성화 (특히 VPN, 광고 차단)

## 신분증 준비

- **여권** (가장 권장 — 영문 이름 포함)
- 국제 운전면허증 (영문)
- 정부 발행 영문 ID

**주의**: 한국 주민등록증이나 한국 운전면허증은 영문이 아니므로 인정되지 않을 수 있습니다. **여권을 강력히 권장**합니다.

## 시간 관리 팁

120분에 60문항 — 문항당 평균 **2분**입니다.

### 추천 페이스 배분

| 구간 | 시간 | 전략 |
|------|------|------|
| 1-20번 | 40분 | 기본 개념 문제 빠르게 |
| 21-40번 | 40분 | 시나리오 문제 신중하게 |
| 41-60번 | 30분 | 남은 문제 + 어려운 문제 |
| 리뷰 | 10분 | Mark for Review 문항 재확인 |

### 핵심 전략

- **모르는 문제는 Mark for Review** — 일단 넘기고 나중에 돌아오기
- **선택지 소거법** — 확실히 틀린 답 먼저 제거
- **시나리오 문제는 질문을 먼저 읽기** — 긴 시나리오에서 핵심만 파악
- **시간 감시** — 30문항 소화 시 60분 이내면 적정 페이스

## 한국 응시 시 주의사항

### 시차

- ProctorFree는 UTC 기준 — 한국 시간(KST = UTC+9) 확인
- 시험 예약 시 시간대 설정 확인 필수
- 새벽 시간대 예약 시 피로도 고려

### 프록터링 시간대

- 프록터링 담당자가 배정되는 시간대를 확인하세요
- 한국 비즈니스 시간(09:00-18:00 KST)에 응시하면 대기 시간이 짧습니다

### 네트워크

- VPN은 반드시 끄세요 — ProctorFree가 차단할 수 있습니다
- 회사 네트워크의 방화벽이 ProctorFree를 차단할 수 있으니 사전 확인하세요
- 모바일 핫스팟은 백업으로 준비하되, 유선 연결을 우선 사용하세요

## 시험 중 기술적 문제 대응

### 인터넷 끊김

1. 당황하지 마세요 — 대부분 자동 재연결됩니다
2. 브라우저를 닫지 마세요
3. 네트워크 복구 후 시험이 이어집니다
4. 타이머는 계속 흐르므로 빠른 복구가 중요합니다

### 브라우저 크래시

1. 시험 링크를 다시 열면 진행 상태가 복원됩니다
2. 답변은 서버에 자동 저장됩니다
3. ProctorFree 지원팀에 채팅으로 즉시 문의하세요

### 프록터 연결 문제

1. 대기 시간이 15분 이상이면 채팅 지원 요청
2. 시험 시작 전 프록터 연결이 안 되면 일정 재조정 가능
