## Claude Partner Network란?

Claude Partner Network는 Anthropic의 공식 파트너 프로그램입니다. 가입하면 CCA-F 시험 무료 응시, 학습 자료 접근, 파트너 전용 지원 등의 혜택을 받을 수 있습니다.

## 가입 절차

### 개인 가입

1. Anthropic 공식 사이트에서 Partner Network 페이지 방문
2. 이메일, 이름, 소속 정보 입력
3. 약관 동의 후 가입 완료
4. Skilljar 플랫폼 접근 권한 부여
5. CCA-F 시험 등록 가능

### 기업 가입

1. 기업 담당자가 파트너 신청서 제출
2. Anthropic 파트너 팀의 검토 및 승인
3. 기업 단위 파트너 계정 생성
4. 소속 직원들에게 시험 응시 권한 배포

## 우리 회사 파트너 여부 확인

사내에서 Claude Partner Network 가입 여부를 확인하려면 IT 부서 또는 AI 관련 부서에 문의하세요.

### 이메일 템플릿

```
제목: Claude Partner Network 가입 여부 확인 요청

안녕하세요,

Anthropic의 Claude Certified Architect - Foundations (CCA-F)
자격증 취득을 위해 Claude Partner Network 가입 여부를
확인하고 싶습니다.

확인 사항:
1. 우리 회사가 Claude Partner Network에 가입되어 있는지
2. 가입되어 있다면, 직원이 CCA-F 시험에 응시할 수 있는
   Skilljar 접근 권한을 받을 수 있는지
3. 미가입 상태라면, 가입 절차를 진행할 담당 부서/담당자

감사합니다.
```

## 무료 응시 혜택

- **최초 5,000명** 무료 응시 기회 제공
- 파트너 네트워크 가입만으로 무료 응시 가능
- 무료 응시 기회 소진 후 유료 전환 예정
- 개인 가입도 무료 응시 대상에 포함

## 기업 내 도입 제안

CCA-F 자격증 취득을 팀/부서 단위로 추진하려면 다음 포인트를 강조하세요.

| 이점 | 설명 |
|------|------|
| 역량 표준화 | 팀원의 AI 아키텍처 역량을 객관적으로 검증 |
| 비용 효율 | 현재 무료 — 비용 부담 없이 시작 가능 |
| 파트너 혜택 | 네트워크 가입 시 추가 학습 자료 및 지원 |
| 커리어 가치 | Anthropic 공식 인증으로 직원 역량 개발 |
| 경쟁력 확보 | AI 도입 프로젝트에서 인증 파트너로서 신뢰도 향상 |
