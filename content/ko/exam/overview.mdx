## Claude란?

Claude는 Anthropic이 개발한 AI 어시스턴트입니다. 안전성과 유용성을 핵심 가치로 설계되었으며, 대화형 AI를 넘어 에이전트, 코드 생성, 엔터프라이즈 통합 등 다양한 영역에서 활용됩니다.

### Claude 제품 생태계

| 제품 | 설명 |
|------|------|
| Claude.ai | 웹/모바일 대화형 인터페이스 |
| Claude API | 개발자용 API (Messages API) |
| Claude Code | CLI 기반 에이전틱 코딩 도구 |
| Claude for Enterprise | 엔터프라이즈 배포 솔루션 |

## CCA-F 정의 및 도입 배경

**CCA-F (Claude Certified Architect - Foundations)**는 Anthropic의 공식 자격증 프로그램의 첫 번째 단계입니다. Claude를 활용한 아키텍처 설계, 에이전트 구축, 워크플로우 자동화에 대한 기초 역량을 검증합니다.

### 도입 배경

- Anthropic의 **$100M+ 투자**를 포함한 파트너십 전략의 일환
- Claude Partner Network를 통한 생태계 확장
- AI 아키텍트 역할의 표준화 및 전문성 검증 수요
- 엔터프라이즈 고객의 파트너 역량 인증 요구

## 자격증 스택 로드맵

현재 CCA-F(Foundations)만 운영 중입니다. 추가 자격증은 공식 발표 시 업데이트됩니다.

| 자격증 | 대상 | 난이도 | 상태 |
|--------|------|--------|------|
| CCA-F (Foundations) | 솔루션 아키텍트 | 입문 | **운영 중** |
| 추가 자격증 | - | - | 미정 (공식 발표 시 업데이트) |

## 타겟 페르소나

공식 Exam Guide에 따르면, 이상적인 후보자는 **프로덕션 애플리케이션을 Claude로 설계·구현하는 솔루션 아키텍트**이며, 다음과 같은 실무 경험이 있는 분입니다:

- Claude Agent SDK를 사용한 에이전틱 애플리케이션 구축 (멀티에이전트, 서브에이전트, 도구 통합)
- CLAUDE.md, Agent Skills, MCP 서버로 Claude Code를 팀 워크플로우에 맞춤 구성
- MCP 도구/리소스 인터페이스 설계
- JSON Schema, few-shot 예시를 활용한 구조화된 출력 프롬프트 엔지니어링
- 컨텍스트 윈도우 관리 및 에스컬레이션/신뢰성 판단

**권장 경험**: Claude API, Agent SDK, Claude Code, MCP를 사용한 **6개월 이상**의 실무 경험

## 타 AI 자격증과의 비교

| 자격증 | 발행사 | 초점 | CCA-F와의 차이 |
|--------|--------|------|---------------|
| AWS ML Specialty | AWS | 클라우드 ML 전반 | CCA-F는 Claude 특화 |
| Google Cloud AI | Google | GCP AI 서비스 | CCA-F는 에이전트 아키텍처 중심 |
| Azure AI Engineer | Microsoft | Azure AI 서비스 | CCA-F는 프롬프트 엔지니어링 비중 높음 |
| CCA-F | Anthropic | Claude 아키텍처 | 에이전트 설계 + MCP 통합에 특화 |

CCA-F는 특정 클라우드 플랫폼이 아닌 **Claude 생태계의 아키텍처 설계 역량**에 집중한다는 점에서 차별화됩니다.
