## 시험 형식

| 항목 | 내용 |
|------|------|
| 문항 수 | **60문항** (객관식, 정답 1개 + 오답 3개) |
| 시험 시간 | **120분** (2시간) |
| 감독 방식 | **ProctorFree** 온라인 감독 |
| 응시 방식 | 원격 온라인 (자택/사무실) |
| 시험 언어 | **영어** |
| 시나리오 | 6개 중 **4개가 랜덤 선택**되어 출제 |
| 오답 패널티 | 없음 — 미응답보다 추측이 유리 |

## 합격 기준

- **합격 점수**: 720 / 1000 (스케일드 스코어)
- 스케일드 스코어는 여러 시험 양식 간 난이도 차이를 보정합니다
- 미응답 문항은 오답 처리됩니다 — 추측이라도 답을 선택하세요

## 비용

- **시험 응시료**: **$99** (정가)
- **무료 응시**: Claude Partner Network 소속 직원 최초 **5,000명** 무료
- 재시험 정책: 불합격 시 일정 기간 후 재응시 가능

### 준비 코스 (무료)

Anthropic Academy의 **13개 준비 코스**가 Skilljar에서 무료로 제공됩니다. 파트너 접근 없이도 수강 가능합니다.

### 시나리오 구조

시험은 6개 시나리오 중 **4개가 랜덤 선택**되어 출제됩니다. 모든 문제는 현실적인 프로덕션 상황을 제시하는 시나리오에 기반합니다.

### 한국 카드 결제 주의사항

- Partner Network 소속이면 무료 (5,000명 한정)
- 정가 $99 결제 시 해외 결제 가능한 카드 필요
- Skilljar 플랫폼 기준 결제 시스템 적용

## 등록 방법

1. **Claude Partner Network** 가입 (무료)
2. **Skilljar** 플랫폼에서 시험 등록
3. 이메일로 응시 링크 수신
4. 원하는 시간에 ProctorFree로 응시

**공식 링크:**
- Claude Partner Network: Anthropic 공식 사이트에서 가입
- 시험 등록: Skilljar 플랫폼
- Exam Guide PDF: 공식 시험 가이드 문서

## 결과 통보

- **소요 시간**: 시험 완료 후 **약 2영업일**
- **결과 내용**: 합격/불합격 + 도메인별 점수 분석
- **디지털 배지**: 디지털 배지 발급 (LinkedIn 프로필에 등록 가능)

### 도메인별 점수 분석

결과 리포트에서 5개 도메인별 강점/약점을 확인할 수 있습니다. 불합격 시 약한 도메인을 집중적으로 보완하여 재시험에 대비하세요.
