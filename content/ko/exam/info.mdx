## 시험 형식

| 항목 | 내용 |
|------|------|
| 문항 수 | **60문항** (객관식 single/multiple choice) |
| 시험 시간 | **120분** (2시간) |
| 감독 방식 | **ProctorFree** 온라인 감독 |
| 응시 방식 | 원격 온라인 (자택/사무실) |
| 시험 언어 | **영어** |

## 합격 기준

- **합격 점수**: 720 / 1000 (스케일드 스코어)
- 단순 정답률이 아닌 문항 난이도 가중 점수입니다
- 72% 이상의 정답률이 대략적인 목표이나, 문항 난이도에 따라 달라질 수 있습니다

## 도메인별 출제 비중

| 도메인 | 비중 |
|--------|------|
| Agentic Architecture & Orchestration | 27% |
| Claude Code Configuration & Workflows | 20% |
| Prompt Engineering & Structured Output | 20% |
| Tool Design & MCP Integration | 18% |
| Context Management & Reliability | 15% |

## 비용

- **시험 응시료**: 파트너 네트워크 가입 시 **무료** (최초 5,000명)
- 이후 유료 전환 예정 (구체적 금액 미정)
- 재시험 정책: 불합격 시 일정 기간 후 재응시 가능

### 한국 카드 결제 주의사항

- 현재 무료이므로 결제 이슈 없음
- 유료 전환 시 해외 결제 가능한 카드 필요 가능성
- Skilljar 플랫폼 기준 결제 시스템 적용

## 등록 방법

1. **Claude Partner Network** 가입 (무료)
2. **Skilljar** 플랫폼에서 시험 등록
3. 이메일로 응시 링크 수신
4. 원하는 시간에 ProctorFree로 응시

**공식 링크:**
- Claude Partner Network: Anthropic 공식 사이트에서 가입
- 시험 등록: Skilljar 플랫폼
- Exam Guide PDF: 공식 시험 가이드 문서

## 결과 통보

- **소요 시간**: 시험 완료 후 **약 2영업일**
- **결과 내용**: 합격/불합격 + 도메인별 점수 분석
- **디지털 배지**: Credly를 통한 디지털 배지 발급
- **LinkedIn 연동**: 배지를 LinkedIn 프로필에 등록 가능

### 도메인별 점수 분석

결과 리포트에서 5개 도메인별 강점/약점을 확인할 수 있습니다. 불합격 시 약한 도메인을 집중적으로 보완하여 재시험에 대비하세요.
