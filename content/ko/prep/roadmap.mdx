## 12주 정석 로드맵

시간적 여유가 있다면 12주 로드맵을 추천합니다. 각 도메인을 깊이 있게 학습하고, 충분한 연습 문제를 풀 수 있습니다.

| 주차 | 주제 | 핵심 활동 |
|------|------|----------|
| 1-2 | 기초 다지기 | Claude 생태계 이해, Exam Guide PDF 정독, 도메인 구조 파악 |
| 3-4 | Agentic Architecture (27%) | 에이전트 루프, 오케스트레이션, 멀티에이전트 패턴 |
| 5-6 | Claude Code & Prompt Engineering (40%) | CLAUDE.md, 슬래시 커맨드, JSON Schema, few-shot |
| 7-8 | Tool Design & MCP + Context (33%) | MCP 서버, 툴 설계, 컨텍스트 관리 |
| 9-10 | 시나리오 분석 | 6개 시나리오 워크스루, 도메인 매핑 연습 |
| 11 | 모의시험 & 복습 | 전체 모의시험, 오답 분석, 약점 보강 |
| 12 | 최종 점검 | 치트시트 복습, 시험 환경 테스트, 응시 |

## 2~4주 단축 루트

이미 Claude를 사용해본 경험이 있거나 AI 분야에 익숙한 분을 위한 단축 루트입니다.

### 2주 루트 (하루 2-3시간)

| 일차 | 활동 |
|------|------|
| Day 1-2 | Exam Guide PDF 정독 + 도메인 개요 전체 읽기 |
| Day 3-4 | Agentic Architecture 심화 (27%로 최고 비중) |
| Day 5-6 | Claude Code + Prompt Engineering |
| Day 7-8 | MCP + Context Management |
| Day 9-10 | 시나리오 분석 6개 전체 |
| Day 11-12 | 치트시트 복습 + 모의시험 |
| Day 13-14 | 오답 분석 + 시험 응시 |

### 4주 루트

2주 루트를 기본으로, 각 도메인에 추가 1일씩 배정하여 여유 있게 학습합니다.

## Anthropic Academy 코스

Anthropic이 공식 제공하는 학습 코스입니다. Skilljar 플랫폼에서 수강 가능합니다.

| 코스 | 예상 소요 시간 | 관련 도메인 |
|------|-------------|-----------|
| Introduction to Claude | 1시간 | 전체 |
| Prompt Engineering Fundamentals | 2시간 | Prompt Engineering |
| Tool Use & MCP | 2시간 | Tool Design & MCP |
| Agentic Workflows | 3시간 | Agentic Architecture |
| Claude Code Mastery | 2시간 | Claude Code |

**추천 수강 순서**: Introduction → Prompt Engineering → Tool Use → Agentic Workflows → Claude Code

- Claude Code CLI 설치 → 실제 코딩 에이전트 체험
- MCP 서버 구축 실습 → 로컬 환경에서 MCP 서버 연동
- CLAUDE.md 작성 연습 → 프로젝트 규칙 정의 실습

## 학습 전략 팁

1. **비중 기반 시간 배분**: Agentic Architecture(27%)에 가장 많은 시간 투자
2. **시나리오 → 도메인 매핑 연습**: 각 시나리오가 어떤 도메인과 관련되는지 익히기
3. **안티패턴 숙지**: "하지 말아야 할 것"이 자주 출제됨
4. **영어 용어 익숙해지기**: 시험이 영어이므로 핵심 용어의 영문 표현 암기
5. **실습 병행**: 이론만이 아닌 Claude Code 직접 사용 경험이 도움됨
