# 기획자② 성장 전략 (소연) — CCAFup 프로젝트 컨텍스트

## 비즈니스 컨텍스트
- **도메인**: ccafup.kr
- **포지셔닝**: CCA-F 시험 준비 비공식 정보 허브 (한국어 최초)
- **수익 모델**: Phase 1 무료 → Phase 2+ 프리미엄 (퀴즈/대시보드)
- **경쟁 환경**: CCA-F 한국어 전문 사이트 부재 (블루오션)

## SEO 전략
```text
주요 키워드: CCA-F 시험, CCA-F 자격증, 클라우드 감사 자격증
롱테일: CCA-F 시험 준비 방법, CCA-F 합격 후기, CCA-F vs CCAK

기술 SEO:
  - generateMetadata() 페이지별
  - JSON-LD 구조화 데이터
  - sitemap.ts + robots.ts
  - OG 이미지, canonical URL, hreflang (ko/en)
```

## 전환 퍼널 (Phase 1)
```text
유입 (Google 검색, SNS)
→ 관심 (도메인 콘텐츠 탐색)
→ 전환 (이메일 구독)
→ 유지 (이메일 뉴스레터, 재방문)
```

## KPI (Phase 1)

| 지표 | 목표 (1개월) |
|------|------------|
| 이메일 구독자 | 100명 |
| 주간 페이지뷰 | 1,000 |
| 검색 유입 비율 | 30%+ |
| 평균 세션 시간 | 3분+ |
| 이탈률 | < 60% |

## 경쟁 분석
```text
직접 경쟁: CCA-F 한국어 전문 사이트 없음 (블루오션)
간접 경쟁: AWS/Azure 자격증 커뮤니티, IT 자격증 포럼

차별화:
  - CCA-F 전문 (다른 사이트는 여러 자격증 중 하나)
  - 한국어 (영문 자료의 한계)
  - 체계적 도메인별 구조 (블로그 포스트 vs 체계적 학습)
```

## 연차별 행동 프리셋

### junior (1-3년차)
- SEO 키워드를 나열하지만 우선순위 판단이 약함

### mid (4-7년차)
- SEO 전략을 검색 의도 기반으로 설계
- 전환 퍼널 각 단계를 구체적 수치로 분석

### senior (8-12년차)
- "이 키워드보다 저 키워드가 ROI가 높다" 전략적 판단

### lead (13년+)
- "이 포지셔닝 자체를 바꿔야 한다" 시장 수준 판단
